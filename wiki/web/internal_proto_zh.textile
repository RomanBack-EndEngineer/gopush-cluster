<h3>Terry-Mao/gopush-cluster Web http协议文档</h3>
Web内部接口文档，用于推送消息、节点管理等

<h3>接口汇总</h3>
(head). | 接口名 | URL | 访问方式 |
| "推送消息":AdminPush   | /admin/push     | POST |
| "添加节点":AdminNodeAdd| /admin/node/add | POST |
| "删除节点":AdminNodeDel| /admin/node/del | POST |

<h3>公共返回码</h3>

(head). | 错误码 | 描述 |
| 0 | 成功 |
| 65534 | 参数错误 |
| 65535 | 内部错误 |

<h3>基本返回结构</h3>
<pre>
{
    "ret": 0,  //错误码
    "msg": "ok",
    "data": {….}//如果有数据返回，则有data，无数据返回就只有前面的ret和msg
}
</pre>

<h3>推送消息</h3>
 * 请求参数

(head). | 参数 | 类型 | 描述 |
| key    | string | 订阅key |
| mid    | int64  | 消息ID  |
| expire | int64  | 消息过期时间，单位：秒(s)|
注: 消息体存放到body中，以上参数为URL参数

 * 返回码

(head). | 错误码 | 描述 |
| 1001 | 没有找到key对应的节点 |
<pre>
{
    "ret": 0,
    "msg": "ok"
}
</pre>

<h3>添加节点</h3>
 * 请求参数

(head). | 参数 | 类型 | 描述 |
| node | string | 节点名称 |

 * 返回码

<pre>
{
    "ret": 0,
    "msg": "ok"
}
</pre>

<h3>删除节点</h3>
 * 请求参数

(head). | 参数 | 类型 | 描述 |
| node | string | 节点名称 |

 * 返回码

<pre>
{
    "ret": 0,
    "msg": "ok"
}
</pre>
