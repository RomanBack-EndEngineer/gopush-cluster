<h3>Terry-Mao/gopush-cluster Web http协议文档</h3>
Web外部接口文档，用于获取Subcribe节点等

<h3>接口汇总</h3>
(head). | 接口名 | URL | 访问方式 |
| "获取订阅节点":ServerGet | /server/get | GET |

<h3>公共返回码</h3>

(head). | 错误码 | 描述 |
| 0 | 成功 |
| 65534 | 参数错误 |
| 65535 | 内部错误 |

<h3>基本请求结构</h3>
<pre>
{    "ret": 0,  //错误码    "msg": "ok",    "data": {….}//如果有数据返回，则有data，无数据返回就只有前面的ret和msg}
</pre>

<h3>获取订阅节点</h3>
 * 请求参数

(head). | 参数 | 类型 | 描述 |
| key | string | 订阅key |

 * 返回参数说明

(head). | 参数 | 类型 | 描述 |
| server | string | 返回的可用于订阅的地址 |

 * 返回码

(head). | 错误码 | 描述 |
| 1001 | 没有找到key对应的节点 |
<pre>
{    "ret": 0,    "msg": "ok"
    "data":{
	"server": "127.0.0.1:8080"  //IP:Port
    }}	
</pre>
